Project CARDESIGN (my first Landing Page)

This project, CarDesign, is my very first landing page built with pure HTML and CSS, inspired by a Figma concept. It showcases a luxury car brand, featuring a sleek layout with branding elements, counters, and a reservation section for high-end vehicles. Although not perfect, I intentionally kept it as it is, because it represents my starting point in web design. This project required a lot of time and dedication, but it allowed me to progress quickly in design fundamentals and apply responsive techniques for the first time.

MoonBank

MoonBank is a modern digital banking landing page designed and built with HTML, JS and Tailwind CSS, and responsive best practices. The project focuses on showcasing online bank with features such as instant transfers, multi-currency wallets, secure transactions, and community-driven initiatives. The design highlights a clean, professional aesthetic with sections like hero, features, trust, achievements, giving back, and contact, each carefully structured for clarity and conversion. This project allowed me to refine my skills in responsive layouts, UI consistency, and the use of branding elements to create a realistic fintech concept.
